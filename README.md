# BarsaPad
This is my Hackpad submission repository for Hackclub x AMD Blueprint.

# Features
* Sandwich mount design
* EC11 Rotary Encoder
* 3 Keys
* 0.91" OLED display

# CAD 
The plate and bottom case will be screwed together by 4 M3 screws with use of heatset inserts positioned on the bottom case and clearance holes located in the corners of the plate. The models were designed in Fusion360.

# BOM
* 1x Seeed XIAO RP2040
* 3x through-hole 1N4148 Diodes
* 3x MX-Style switches
* 1x EC11 Rotary encoders
* 1x 0.91 inch OLED display
* 3x white blank DSA keycaps
* 4x M3x16mm screws
* 4x M3x5mx4mm heatset inserts
* 1x Case (2 printed parts)
